# **App Name**: PrepStack AI

## Core Features:

- Roadmap Generation: Generate personalized learning roadmaps based on user's career goals and experience level, including optimal skills to learn.
- Practice Module Customization: Tailor practice modules using AI to focus on user's weaknesses and learning preferences for efficiency and content curation.
- Smart Resource Recommendation: AI-powered tool to analyze user progress and recommend resources like resume templates and cheat sheets that fill the learning gaps.
- Adaptive Test Engine: Provide adaptive testing that adjusts difficulty based on real-time performance, creating precise assessment experiences.
- User Authentication: Secure user accounts to save progress, preferences and access personalized roadmaps.
- Interactive Learning Modules: Display comprehensive explanations, code examples, and practice questions for each topic.
- Firestore Contact Form Integration: Contact form to submit feedback, saving the data in Firebase Firestore for analysis.

## Style Guidelines:

- Primary color: Neon Blue (#00E5FF) for a vibrant, tech-focused aesthetic.
- Background color: Deep Black (#0B0F19) providing a high contrast backdrop.
- Accent color: Neon Purple (#9D4EDD) to highlight key interactive elements and create depth.
- Body and headline font: 'Inter' for a modern and clean look. Note: currently only Google Fonts are supported.
- Use a set of consistent, minimalist icons with a neon glow effect to represent various modules and resources.
- Employ glassmorphism cards with soft glow effects and floating 3D shapes to create an interactive UI experience.
- Implement subtle hover animations, smooth page transitions using framer-motion for a high level of visual polish.
